## Vista del proyecto 🚀

Mira **[ToDo-App](https://julian1david-react-todo.netlify.app/)** para conocer como funciona el proyecto.

### Instalación 🔧

```
git clone https://github.com/julianf-dev/react-todo-test
```

```
    npm install
    yarn install
```

```
    npm run dev
    yarn run dev
```
